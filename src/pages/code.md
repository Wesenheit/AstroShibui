As part of my day-to-day work I (co)authored several software projects. Those include:

1. [rixa](https://github.com/Wesenheit/rixa) - PMIx-based PyTorch launcher. Native Slurm/MPI integration, no need to specify
master IP address/port.
2. [Verona](https://github.com/Wesenheit/Verona) - Julia-based, GPU-accelerated computational-fluid dynamics code. Utilizing MPI to
support multi-node, large-scale simulations for Special-relativistic hydrodynamics (SRHD).
3. [Lume](https://github.com/Wesenheit/Lume) - Visualizing CPU/GPU workload with [Connection Machine blinking lights](https://www.youtube.com/watch?v=KmiFBI1f2o4).
